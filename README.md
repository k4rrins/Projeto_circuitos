# Projeto_circuitos
Projeto de circuitos digitais desenvolvido no Logisim. Consiste em um minigame no qual o usuário tenta adivinhar um número aleatório de 0 a 15 em binário. LEDs RGB indicam a proximidade do palpite, displays exibem a conversão do número e um LED sinaliza o acerto, além de outros componentes internos responsáveis pela lógica do sistema.
